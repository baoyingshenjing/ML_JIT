2. Implement model.py with the EXACT JiT architecture:
   - BottleneckPatchEmbed: Conv2d(in->bottleneck_dim, kernel=patch_size) -> Conv2d(bottleneck_dim->hidden, 1x1)
   - TimestepEmbedder: sinusoidal -> SiLU -> Linear
   - LabelEmbedder: nn.Embedding(num_classes+1, hidden)
   - Attention: QKV projection, QK RMSNorm, RoPE, use F.scaled_dot_product_attention
   - SwiGLUFFN: w12(x) -> chunk -> SiLU(gate)*up -> w3
   - JiTBlock: pre-norm RMSNorm, attention+MLP with adaLN (6*hidden modulation)
   - JiT: stacks blocks, in-context class tokens inserted at depth//3 position, fixed sincos position embed
   - FinalLayer: RMSNorm + adaLN modulation + Linear(hidden->patch*patch*3)
   - unpatchify: reshape tokens back to image
   - Factory functions: jit_b_16(), jit_b_32(), jit_l_16(), jit_h_16()
   
   EXACT SPECS (read from reference /home/quank/JiT/model_jit.py):
   - JiT-B/16: depth=12, hidden=768, heads=12, bottleneck=128, in_context_len=32, in_context_start=4
   - JiT-B/32: depth=12, hidden=768, heads=12, bottleneck=128, in_context_len=32, in_context_start=4, patch=32
   - JiT-L/16: depth=24, hidden=1024, heads=16, bottleneck=128, in_context_len=32, in_context_start=8
   - JiT-H/16: depth=32, hidden=1280, heads=16, bottleneck=256, in_context_len=32, in_context_start=10
   - Dropout only in the last quarter of blocks (depth//4*3 to depth)
   - Use torch.compile on JiTBlock.forward and FinalLayer.forward
